# Hattab

## Requirement

To run this application locally you need :

**Node.js** and **NPM**

Available on https://nodejs.org/.
To update NPM:

```sh
npm install --global npm
```

**Emulator**

Use Android studio for an android emulator or Xcode for an ios simulator (macOS).

## Use

To run the application, open a terminal at the root project. To install all the dependencies :

```sh
npm install
cd ios && pod install
```

Before running the application, you need to rename the .env.example file in .env and fill the file.
Then, run the application by running the command :

```sh
react-native start
```
